# 🌍 Travel Agency Website

A modern, responsive multi-page travel agency website built using React and Tailwind CSS. This project demonstrates clean UI/UX design, responsive layouts, and essential travel website features.

---

## 🚀 Objective

To design and develop a professional travel agency website with multiple pages, focusing on responsiveness, user experience, and structured layout design.

---

## 📄 Pages Included

- Home
- About Us
- Services / Destinations
- Packages
- Contact Us

---

## ✨ Features

- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- 🧭 Navigation bar and footer on all pages
- 🎯 Clean and modern UI/UX
- 🏝️ Destination listing with search/filter functionality
- 💰 Travel packages with pricing details
- 🖼️ Image gallery / slider
- ⭐ Testimonials section
- 📩 Contact form with validation
- 🔘 Call-to-action buttons (Book Now, Enquire Now)
- 🎬 Scroll animations using AOS

---

## 🛠️ Tech Stack

- ⚛️ React (v19)
- 🎨 Tailwind CSS (v4)
- 🎞️ AOS (Animate On Scroll)
- 🎯 React Icons
- 🔗 React Scroll
- ⚡ Vite
